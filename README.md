# re-bd-map

大数据课程项目-数据可视化部分
+ 运行前需确认百度地图ak、路网信息及api可用性

## 技术栈
+ vue.js 2.x
+ [vue-baidu-map](https://dafrok.github.io/vue-baidu-map/#/zh/index) 等价于[百度地图API JSv2](http://lbsyun.baidu.com/index.php?title=jspopular)
+ element-ui
+ axios
